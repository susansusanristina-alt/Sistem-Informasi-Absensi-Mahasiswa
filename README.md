# UAS Sistem Informasi Absensi Mahasiswa

## Deskripsi Proyek

Sistem Informasi Absensi Mahasiswa merupakan aplikasi berbasis web yang dibuat sebagai proyek Ujian Akhir Semester (UAS) Mata Kuliah Basis Data. Aplikasi ini digunakan untuk mengelola data mahasiswa dan data absensi secara terkomputerisasi sehingga proses penyimpanan, pencarian, dan pengelolaan data menjadi lebih mudah dan efisien.

## Tujuan Proyek

- Menerapkan konsep perancangan basis data relasional.
- Mengimplementasikan database menggunakan MySQL.
- Membangun aplikasi web sederhana menggunakan PHP.
- Menerapkan operasi CRUD (Create, Read, Update, Delete).
- Mengimplementasikan query SQL, JOIN, VIEW, INDEX, DCL, dan TRANSACTION.

## Cara Menjalankan Program

1. Jalankan Apache dan MySQL pada XAMPP.
2. Import seluruh file SQL ke phpMyAdmin.
3. Simpan folder proyek pada folder **htdocs**.
4. Jalankan aplikasi melalui browser.

Contoh:

```
http://localhost/sistem_absensi/app/index.php
```
